## Prerequisites

*  Docker
*  Docker Compose

## Installation

```sh
docker-compose up -d
```

## Used Dependencies
* Spring Boot Web
* Spring Boot Artemis
* Spring Boot Actuator
* Lombok
* Dev Tools
* Swagger (Springfox)

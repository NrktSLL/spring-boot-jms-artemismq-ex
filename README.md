>ArtemisMQ example with jms.

## Prerequisites

*  Docker <B>20.10.0</B> or more recent
*  Docker Compose <B>1.27.4</B> or more recent

## Installation

```sh
docker-compose up -d
```

## Used Dependencies
* Spring Boot Web
* Spring Boot Artemis
* Spring Boot Actuator
* Lombok
* Dev Tools
* Swagger (Springfox)
